# N2O Gaming Landing Page

Gaming landing page built from Figma community design.

## Technologies
- HTML5 / CSS3 / Vanilla JS
- Responsive: 1280px / 768px / 375px
- BEM methodology
- CSS Custom Properties

## Features
- Custom video player with play/pause
- Testimonials slider with autoplay
- FAQ interactive tabs
- Burger menu for mobile
